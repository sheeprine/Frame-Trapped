Patch and Change Notes

Current version: 2.6 (rev 30)

Donate to the FT Project

As you may have guessed, this is not my day job. I'm a developer by day and this is my hobby by night. It is my own free time that I spend to create this and I do it for the love of gaming and the community behind it. Versions

2.6

  • Revamped Library Layout- Fighter selection now docked at the top. Moves list is simpler, break down now stretches out on the right.
  • Time Line Playback Improvements - Multi-threaded playback now means less hanging on the UI and live updating to show which command is currently being playd.
  • Bug fixes on Steamworks - Don't want to test it full screen? Plans are to embed Street Fighter fully with in Frame Trapped so that you can test your moves in an isolated environment without needing to switch windows.
  • Library to Time Line Improved - There are now Player One and Player Two buttons next to the inputs in the Move Breakdown header bar. Clicking these will put the commands into the respective time line queues.
  • Save/Append/Open separated into Player One and Two - Does what it says on the tin.

2.5

  • Insert from library - Now you can add frame data specific inputs from the library into the combo trainer time line after your current item.
  • Resolution Set - change the game view resolution on the fly to fit your preferences. It will scale to stretch when you resize FT.
  • Automatic Keybinds - no need to configure anything for keyboards: all the keybinds are loaded automagically from the config files in your Documents/CAPCOM folder.
  • Automatic Steam loader - we now load SSF4 using the Steam protocol. You don't need to specify your game location or Steam location (Steam version support only for now, guys, make sure you claim your SSF4 keys on Steam if you can!)
  • Steam works memory fix - We use SigScan to find the memory addresses needed in game now, this should mean a little future proofing for Steam based updates. It's experimental so we'll keep a close eye on this one.
  • Repeater functionality - Loop your time lines in sync between to a max of 99 repeats. Don't worry about matching the time lines frame count - FT fills in the blanks.
  • Player Two Support! - Finally, on of my favourite additions. You can setup a second time line and play the player two Human in training to test out precise time lines between two different fighters. Make sure set full key binds for both players.
  • ClickOnce install - now using Microsoft's ClickOnce installer I can send you updates automatically; no need to constantly download the tool with new releases.

2.4 [Offline release]

  • Insert from library - Now you can add frame data specific inputs from the library into the combo trainer time line after your current item.
  • In View Street Fighter Embedding - Embedding SSFIV.exe inside the actual Combo Trainer view means you won't have to switch screens when performing combos
  • Fullscreen switching support - The perks of the way we embed the game also means that Fullscreen will work, even to playback combos from the timeline.

2.3

  • 32 bit support
  • Steam works fixes - after the patch from GFWL we needed to fix it! It works now.
  • Time line items from Library to ComboTrainer feature - Add time line items using a context menu in the Library
  • Options to set Steam executable location - (obsolete - now automated)
  • Resolution selection to scale embedded SF4 game
